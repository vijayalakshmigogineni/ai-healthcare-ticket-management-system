# Workflow Explanation

1. Client sends email.

2. Gemini AI classifies the communication.

3. Ticket is automatically generated.

4. Agent repository is searched.

5. Best available agent is selected.

6. SLA deadlines are calculated.

7. Dashboard records are generated.

8. Managers can reassign tickets.

9. Agents resolve tickets.

10. Audit logs are generated.

11. Executive summary is produced.
